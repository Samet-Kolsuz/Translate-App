#Translate App
A modern, user-friendly translation application built with React and Redux Toolkit, enabling seamless translation of text across multiple languages using an external translation API.

##Kütüphaneler
- @reduxjs/toolkit
- react-redux
- axios
- react-select
- tailwindcss
